# Hack-CMC-Heineken

Our team participate in Hackathon hackcmc, give the solution for Heineken problem

## 1. Getting Started
```shell  
git clone https://github.com/sonhm3029/Hack-HCMC-Solution.git
cd Hack-HCMC-Solution

conda create --name heineken python=3.9.13
conda activate heineken
pip install -r requirements.txt
```

## 2. Run backend
```shell
python Hack-HCMC-Solution\MVP\Flask-Server-template\app.py
```

## 3. Android APP
Download this APK file [drive](https://drive.google.com/file/d/1dZgT7JMedZP-LIp15Kd4E7XVzgTrqeTK/view?fbclid=IwZXh0bgNhZW0CMTAAAR14SW8fc3R9bciViJtxu6r7JG5R6PM1ymw44VhLd7uMhQ39S1WZ-L4A5GI_aem_G9IF3Rc4VntpvAzsaniwSg)
then, install it on your Android mobile.

## 4. Website

```shell
cd MVP/Vite-ReactJS-Base
yarn
yarn start
```
